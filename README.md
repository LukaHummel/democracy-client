# Contribute

## install dependencies

```
clone PROJECT_ULR
cd democracy
yarn install

```

## start developing UI

### Android

```
cd mobile-ui
yarn android
CMD+M and Change Bundle Location to 127.0.0.1:8088
```

### iOS

```
cd mobile-ui
yarn ios
CMD+M and Configure Bundler Location to Host: 127.0.0.1 & Port: 8088
```
